# Proposal Engine Overview

These screenshots were captured from the live Proposal Engine and show the end-to-end workflow from pipeline intake through proposal assembly.

## Projects

<img src="/img/screenshots/proposal-engine/01-projects.png" alt="Proposal Engine projects view" />

The Projects view gives Ahmad and the team a live operating table of active proposal work.

## Pipeline

<img src="/img/screenshots/proposal-engine/02-pipeline.png" alt="Proposal Engine pipeline view" />

Pipeline organizes opportunities by stage so the team can see what is moving and what is blocked.

## Estimator

<img src="/img/screenshots/proposal-engine/03-estimator.png" alt="Proposal Engine estimator view" />

Estimator turns scope inputs into labor, margin, and pricing outputs for a working proposal draft.

## RFP Analyzer

<img src="/img/screenshots/proposal-engine/04-rfp-analyzer.png" alt="Proposal Engine RFP analyzer" />

RFP Analyzer extracts requirements from client documents into a structured proposal starting point.

## SOW Builder

<img src="/img/screenshots/proposal-engine/05-sow-builder.png" alt="Proposal Engine SOW builder" />

SOW Builder lets the team assemble deliverables, assumptions, and scope language in one flow.

## Spec Sheets

<img src="/img/screenshots/proposal-engine/06-spec-sheets.png" alt="Proposal Engine spec sheets" />

Spec Sheets centralizes product documentation and output-ready technical pages for proposals.

## Settings

<img src="/img/screenshots/proposal-engine/07-settings.png" alt="Proposal Engine settings" />

Settings manages the account-level controls that shape how proposals are generated and presented.

## Proposal Detail

<img src="/img/screenshots/proposal-engine/08-proposal-detail.png" alt="Proposal Engine proposal detail view" />

The proposal detail workspace is where scope, builder output, and final edits come together before handoff.
